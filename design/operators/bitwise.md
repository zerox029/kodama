# Bitwise operators
| Operators | Description         |
| --------- | ------------------- |
| `&`       | Bitwise AND         |
| `\|`      | Bitwise OR          |
| `^`       | Bitwise XOR         |
| `~`       | Bitwise NOT         |
| `<<`      | Bitwise left-shift  |
| `>>`      | Bitwise right-shift |
